## 介绍
Nobita 是一个基于Koa而诞生的一款框架。

[API文档](https://nobitajs.github.io/nobita/)

## 快速入门

```
$ npm install nobita-cli -g
$ nobita-cli init
$ cd 项目名
$ npm install
$ npm run local
```
访问：[http://127.0.0.1:6001](http://127.0.0.1:6001)
